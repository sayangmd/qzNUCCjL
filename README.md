# 项目简介

本系统为基于深度学习YOLOv11的红细胞、白细胞和血小板检测系统。其主要功能模块包括利用YOLOv11算法对医疗图像进行快速、准确的细胞检测，并提供直观的界面展示检测结果。系统整合了专用的YOLO数据集用于模型训练与优化，同时包含用户登录注册界面，保障用户数据安全。技术上，系统采用深度学习技术，具有识别准确率高、实时性好的特点，并提供完整的源码，便于进一步研究和应用拓展。


## 核心功能
- 基于YOLOv11算法对医疗图像中的红细胞、白细胞和血小板进行快速、准确的检测
- 提供直观的界面展示检测结果
- 整合专用YOLO数据集用于模型训练与优化
- 包含用户登录注册功能，保障用户数据安全
- 提供完整的源码，便于进一步研究和应用拓展

## 适用场景
- 医疗机构的血液检测与分析
- 医学研究中的细胞计数与分类
- 医学图像处理与诊断辅助
- 医学教育与培训
- 生物医学研究中的细胞形态学分析

## 技术特点
- 采用深度学习技术，具有识别准确率高的特点
- 实时性好，能够快速处理医疗图像
- 系统架构完整，包含前端界面、后端逻辑和深度学习模型
- 基于YOLOv11算法，结合专用数据集进行优化
- 提供源码，支持二次开发和功能扩展
